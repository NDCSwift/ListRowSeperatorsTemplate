# 📋 List Row Separators — Template
A SwiftUI starter template for the List Row Separators tutorial — follow along from scratch.

---

## 🤔 What this is
This is the starter template for the List Row Separators tutorial. It provides the base project structure so you can follow along with the video and add separator customization yourself. Clone this if you want to build it step by step rather than jumping straight to the completed version.

## ✅ Why you'd use it
- **Follow-along starter** — Pre-configured project so you can focus on learning the separator APIs without setup overhead
- **Separator visibility** — You'll add `.listRowSeparator(.hidden)` to control per-row separator display
- **Separator tinting** — You'll use `.listRowSeparatorTint(_:)` to apply custom colors to dividers

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/ListRowSeperatorsTemplate.git
cd ListRowSeperatorsTemplate
```

### 2. Open in Xcode
Double-click `ListRowSeperators.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- This is the starter template. For the finished project, see the `ListRowSeperatorsCompleted` repo.
- If you see a code signing error, verify Team and Bundle ID are set.

## 📦 Requirements
- Xcode 15+
- iOS 16+
